# go-tput
execute tput by Go.

## Usage
### Reading
| Function | Description |
| ---- | ---- |
| tput.Cols() | Number of columns in the terminal |
| tput.HR() | Print horizontal line |
### Text Effects
| Function | Description |
| ---- | ---- |
| tput.Setaf(i int) | Set foreground color |
| tput.Smul() | Start underlined text |
| tput.Sgr0() | Turn off all attributes |
